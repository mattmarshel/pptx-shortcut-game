# PowerPoint Master

An interactive web-based game to learn 50 PowerPoint shortcuts in one day.

## How to Play

Open `index.html` in any modern browser. No build step or server required.

### Game Modes

- **Learn** — Interactive flashcards with spaced repetition. Rate your recall to schedule reviews.
- **Practice** — Press the actual key combination on your keyboard to test muscle memory.
- **Quiz** — 10-question multiple choice rounds to reinforce recognition.
- **Speed Run** — 60-second timed challenge. How many shortcuts can you nail?

### Features

- Progress tracking (new / learning / mastered) with localStorage persistence
- 50 shortcuts across 9 categories (Alignment, Formatting, Insert, Shapes, View, etc.)
- Searchable reference table with category filters
- Beautiful dark UI with smooth animations
- Fully responsive for mobile and desktop
